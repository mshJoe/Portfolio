# Youssef's Portfolio 💻

A minimalist, terminal-inspired personal portfolio. Built to showcase projects, skills, and academic milestones with a clean, high-performance Matrix/Terminal aesthetic.

## 🚀 Tech Stack
- **Structure:** HTML5
- **Styling:** Tailwind CSS (Utility-first)
- **Logic:** Vanilla JavaScript
- **OS:** Crafted on Fedora Linux

## 🛠 Features
- **Matrix Aesthetic:** Dark-themed terminal interface.
- **Responsive Design:** Optimized for Desktop and Mobile.
- **Dynamic Media Log:** Tracks my "Also Liked" movies/shows.
- **Social Integration:** Live links to GitHub, LinkedIn, and IMDb.
