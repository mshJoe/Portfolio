## ✨ Key Features

*   **🕶️ Matrix Aesthetic:** A dark-themed terminal design with custom teal accents that provides a unique browsing experience.
*   **📱 Fully Responsive:** Clean, intuitive UI optimized for desktop, tablet, and mobile devices using CSS Grid & Flexbox.
*   **🎞️ Dynamic Media Log:** A personalized "Also Liked" section that tracks my favorite movies and shows with custom posters and ratings.
*   **🔗 Social Integration:** Quick-access interaction buttons for IMDb, GitHub, LinkedIn, and Discord.
*   **🛠️ Interactive Forms:** Functional UI elements in the contact section mimicking system execution logs for a professional "Hacker" feel.

## 🚀 Built With

*   **Structure:** HTML5 (Modular architecture).
*   **Styling:** Tailwind CSS (Utility-first framework).
*   **Logic:** Vanilla JavaScript (ES6+).
*   **Environment:** Developed on Fedora Linux.

## 🛠️ Local Setup & Installation

Since this project is built using vanilla web technologies and Tailwind, no complex build tools are required.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mshJoe/Portfolio.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Portfolio
    ```
3.  **Run the application:**
    Simply open the `index.html` file in your preferred web browser.
